# Pro Backend

## Inicializa o projeto
npm init -y

## Dependências de produção
npm install express cors dotenv jsonwebtoken bcryptjs @prisma/client

## Dependências de desenvolvimento (TypeScript e tipos)
npm install -D typescript ts-node @types/express @types/cors @types/jsonwebtoken @types/node @types/bcryptjs prisma

# Pro Mobile

## Inicializa o projeto com Expo (Template TypeScript)
npx create-expo-app@latest . --template expo-template-blank-typescript

## Dependências essenciais para Mapas e Navegação
npx expo install react-native-maps react-native-geocoding expo-location
npm install @react-navigation/native @react-navigation/stack axios @expo/vector-icons

# Iniciar Telas
npm start
